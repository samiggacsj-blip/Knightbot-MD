# 📖 دليل توثيق Matrix WhatsApp Bot by DEV ABOODI 
يوفر هذا المستند نظرة عامة على معمارية البوت، ونظام الإضافات (البلجينز)، والاستخدام المتقدم للمشروع.
## 🧠 المعمارية وهيكلة الملفات (Architecture)
```
Goat-WhatsApp-Bot/
├── plugins/
│   ├── commands/     # جميع ملفات وأوامر البوت
│   └── events/       # جميع أحداث الاستماع التلقائية
├── core/             # ملفات المصادقة، المعالجات، والمحمل الرئيسي
├── auth/             # بيانات جلسة الواتساب المشفرة (Session)
├── dashboard/        # واجهة لوحة تحكم الويب للمتصفح
├── config.json       # إعدادات البوت القابلة للتخصيص
├── bot.js            # نقطة انطلاق وتشغيل البرنامج الأساسية

```
## 🔌 نظام الإضافات (Plugin System)
### 1. هيكلية وصيغة الأوامر (Command Structure)
لإنشاء أمر جديد للبوت، قم بإنشاء ملف داخل مسار مجلد الأوامر plugins/commands/ على النحو التالي:
```js
module.exports = {
  config: {
    name: "ping",
    description: "أمر فحص سرعة الاستجابة",
    aliases: ["p"],
    author: "DEV ABOODI",
    cooldown: 5,
    role: 0,
    category: "utility"
  },
  onCmd: async ({ reply }) => {
    reply("بونج!");
  }
};

```
### 2. هيكلية مستمعي الأحداث (Event Listener Structure)
لإنشاء حدث تلقائي (مثل الترحيب أو المغادرة)، قم بإنشاء ملف داخل مسار مجلد الأحداث plugins/events/:
```js
module.exports = {
  config: {
    name: "welcome",
    author: "Anbuinfosec",
    category: "events"
  },
  onEvent: async ({ api, event }) => {
    // معالجة انضمام الأعضاء الجدد هنا
  }
};

```
## 🌐 لوحة التحكم (Dashboard)
يوفر السكريبت واجهة ويب تفاعلية يمكن الدخول إليها عبر المتصفح:
 * /dashboard - نظرة عامة وعرض إحصائيات النظام الفورية.
 * /login - لوحة تسجيل الدخول الآمنة للمطور.
 * /settings - تفعيل وإغلاق ميزات البوت والتحكم الكامل بالإضافات.
> تم بناء هذه اللوحة باستخدام HTML/CSS + JS بسيطة ومباشرة، بدون الحاجة لأي إطار عمل فرونت إند معقد (No frontend framework).
> 
## 🧩 التوسيع والتطوير (Extending)
 * **إضافة قواعد بيانات:** يمكنك ربط أي قاعدة بيانات جديدة عبر تعديل ملف النواة في core/database.js.
 * **إضافة لغات جديدة:** يمكنك تعريب أو إضافة لغات أخرى عبر مجلد اللغات في /languages.
 * **تخصيص الرسائل:** يمكنك تعديل رسائل الترحيب والمغادرة الافتراضية بسهولة من ملف الإعدادات config.json.
